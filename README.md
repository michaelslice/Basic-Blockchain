# Setup
1. Clone the repo
```
git clone https://github.com/michaelslice/Basic-Blockchain.git
```
2. Download requirements
```
pip install -r requirements.txt
```
3. Start the server
```
python run.py
```
# Example Usage
1. Getting your Wallet Info
```
curl http://127.0.0.1:5000/wallet
```
